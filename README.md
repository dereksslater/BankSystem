# 💰 Sistema Bancario em Java

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos basicos de Programacao Orientada a Objetos (POO) em Java.

O sistema simula uma conta bancaria simples, permitindo realizar operacoes como deposito, saque e visualizacao de dados da conta.

---

## ⚙️ Funcionalidades

* Criar conta bancaria com titular e saldo inicial
* Depositar valores
* Sacar valores com validacao de saldo
* Exibir dados da conta

---

## 🛠️ Tecnologias utilizadas

* Java
* NetBeans
* Maven

---

## 📚 Conceitos praticados

* Classes e Objetos
* Construtores
* Encapsulamento
* Metodos
* Estrutura condicional (if/else)

---

## ▶️ Como executar

1. Abrir o projeto no NetBeans
2. Executar a classe Main
